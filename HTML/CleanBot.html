Our Mission

  • Develop an environmentally conscious robot that can autonomously collect trash
  • Integrate AI/ML models for object detection and classification
  • Simulate real-world terrain handling (beach, grass, concrete, etc.)
  • Build an interactive web dashboard for telemetry data (battery, location, torque, object count)
  • Open-source the design for use in global clean-up initiatives

As this project runs and CleanBot collects more recyclable waste, we hope to partner with local facilities, like redemption centers and recycling facilities, where we can turn in the recyclable waste for money, which we aim to donate to charity. We hope to donate every penny earned by trash collection to charity!
